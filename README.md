# Projeto: Landing Page - NullSector CTF
**Instituição:** SENAI CTTI  
**Disciplina:** Programação Client-Side  
**Data de Apresentação:** 15/06  

## 📑 Sobre o Projeto
Este projeto consiste em uma Landing Page responsiva com o tema de **Segurança Cibernética e simulação de CTF (Capture The Flag)**, desenvolvida como avaliação prática para a matéria de Programação Client-Side. 

O objetivo principal é aplicar os conceitos fundamentais de desenvolvimento web front-end vistos em aula, estruturando uma experiência de usuário (UX) imersiva e dinâmica.

### 🛠️ Tecnologias e Conceitos Aplicados:
* **HTML5:** Estruturação semântica de múltiplas páginas (`index.html` e `desafios.html`).
* **CSS3 Nativo:** Estilização moderna e layout responsivo.
* **Flexbox:** Alinhamento dinâmico de seções, menus e elementos de interface.
* **Interatividade:** Aplicação de efeitos de `hover` e propriedades de `transition` para suavidade visual.
